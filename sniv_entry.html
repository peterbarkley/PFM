Sniv entry

  1. Goal: allow users to enter snivs for a resource
  2. At its most basic, this is a form where people can select a [resource: if they have privileges for more than just themselves], a start time, an end time, and a reason. This is stored for users in the sniv table. For aircraft, it is stored in the plane_unavail table.
  3. For convenience, it would be nice to have an "all day" option where you only select a date. This gets recorded as 0000-2359 on that day.
  4. At some point, it would be nice to enable google calendar-type sniv entry. That's not a priority.
