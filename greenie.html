Greenie board information

  1. Goal: display the status of all active students across all items of interest on one page
  2. Ideally, the pass/fail/marginal cells should be links that pull up the grade sheet for the given event and the student name is a link to the student training summary page.
  3. You should be able to sort the table alphabetically by column by clicking on the column header
  4. Note the color scheme:
    • Green for Complete status, Y Medical, <=1 day since last flight, or P (with no failures or marginals).
    • Red for a N Medical, >=4 days without flying or a cell with a failure.
    • Yellow for 1
    • Gray for scheduled events
    • Purple for incomplete or canceled events
    • I'm tempted to create a "flashing" cell for event where the gradesheet has no graded items and it's >2 hours after scheduled sortie land time.
