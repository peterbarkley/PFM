AGI grade sheet entry

  1. Goal: allow AGIs to select an academic class and input grades for a large number of mids (ideally, automatically generated with those assigned to their class)
  2. This could be done by allowing AGIs to select a quiz/test and then allowing grade entry for all active Mids or displaying all possible graded items across the top. Or, rather than all possible, just all events where there is a class_schedule entry for the current block.

Demo

The row entries loop over all active students (foreach student table entry where student->status!='inactive'). The column entries loop over all class_schedule entries where class_schedule->begin >= consts->current_block_start and class[class_schedule->class_ID]->graded is true.
